# Hub71 Demo Script

Use this for a 1-minute Loom or live walkthrough.

## Script

1. Opening: "Hi, I am Shaurya. This is SHAURYA Automation OS, a working internal operator system I use to turn founder goals into routed AI work, worker prompts, and proof reports."
2. Problem: "Founders have many AI tools, but work still fragments across notes, prompts, tabs, and manual proof. The bottleneck is orchestration, memory, and credible evidence."
3. Command Center: "This local Web Command Center opens and runs proof workflows. It is local-only: no external accounts, paid APIs, GitHub push, or n8n workflow builds."
4. Brain Router: "The Brain Router reads local memory and chooses the pipeline, worker, tools, and next action."
5. Worker Prompt Generator: "The Worker Prompt Generator creates worker-specific prompts so the handoff is repeatable."
6. Hub71 Operator: "The Hub71 Submission Operator packages founder input and local context. The current submission packet is READY 12, DRAFT 0, MISSING_CONTEXT 0."
7. GitHub Proof Builder: "The GitHub Proof Builder separates READY, PARTIAL, and FUTURE proof and creates portfolio-ready reports and checklists."
8. Closing: "This is a working internal system, not a revenue or public SaaS claim yet. The next step is productizing it as an operator OS for founders, solo builders, and AI-heavy teams."
